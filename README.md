# backend

- `python 3.12 + poetry` (prerequisites)
- `bash bootstrap.sh`  

WITH DOCKER  
- `docker-compose up --build`
- `docker-compose up -d`
- `docker-compose down` to stop 

WITHOUT DOCKER
- `redis` (prerequisite)
- `bash start_all.sh`

# frontend

- `node version 20.19.0` (prerequisites)
- `npm install`
- `npm run dev:client`
