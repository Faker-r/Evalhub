  % ── The Problem ──
  \begin{tcolorbox}[sectionbox,title={The Problem}]
    {\fontsize{20}{26}\selectfont\color{bodytext}

    AI language models (LLMs) are deployed through many different
    \textbf{inference providers}---companies that serve the same
    open-source model on different hardware and software stacks.

    \vspace{0.4cm}
    Each provider's implementation can \textbf{subtly alter model
    behaviour}: safety, coherence, correctness, and helpfulness can
    all vary, even for the \emph{exact same underlying model}.

    \vspace{0.4cm}
    \textbf{No standardized, automated framework} currently exists
    to detect and quantify these serving-induced differences at scale.
    }

    \vspace{0.4cm}

    \begin{center}
    \begin{tikzpicture}[

  % ── Why This Matters ──
  \begin{tcolorbox}[sectionbox,title={Why This Matters}]
    {\fontsize{20}{26}\selectfont\color{bodytext}
    \begin{itemize}[leftmargin=1.1cm, itemsep=0.5cm,
                    label={\color{evalgreen}\textbullet}]
      \item \textbf{Enterprises} make multi-million dollar AI
            provider decisions with no objective comparison data
      \item \textbf{Researchers} need reproducible benchmarks for
            rigorous scientific comparison
      \item \textbf{Developers} building products lack a tool to
            verify which provider best fits their use case
      \item \textbf{Safety-critical fields} (healthcare, law)
            depend on providers not degrading model safety
    \end{itemize}
    }

  % ── Evaluation Criteria ──
  \begin{tcolorbox}[sectionbox,title={Evaluation Criteria}]
    {\fontsize{19}{24}\selectfont\color{bodytext}
    Each evaluation run scores a provider on four dimensions:}
    \vspace{0.35cm}

    \begin{tcolorbox}[redft]
      {\fontsize{18}{23}\selectfont
      \textcolor{evalgreen}{\textbf{Safety}} --- Does the provider's deployment respect safety guidelines and refuse harmful outputs?}
    \end{tcolorbox}
    \vspace{0.22cm}

    \begin{tcolorbox}[blueft]
      {\fontsize{18}{23}\selectfont
      \textcolor{evalgreen}{\textbf{Coherence}} --- Is the output logically consistent, well-structured, and on-topic?}
    \end{tcolorbox}
    \vspace{0.22cm}

    \begin{tcolorbox}[greenft]
      {\fontsize{18}{23}\selectfont
      \textcolor{evalgreen}{\textbf{Correctness}} --- Are factual answers and reasoning steps accurate?}
    \end{tcolorbox}
    \vspace{0.22cm}

    \begin{tcolorbox}[orangeft]
      {\fontsize{18}{23}\selectfont
      \textcolor{evalgreen}{\textbf{Helpfulness}} --- Does the response fully address the user's actual need?}
    \end{tcolorbox}
